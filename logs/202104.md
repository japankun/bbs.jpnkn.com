# 2021年4月20日

- ID 表示機能のコードを追加した。
- 管理画面の反映はまだしていない。

### オプションは以下の3つ

- 非表示
　- これまで通り ID は表示されない(デフォルト設定)
- 任意表示
  - メール欄が空の場合 ID を表示する
  - メール欄がある場合は ID:??? と表示する
- 強制表示
　- いかなる場合も ID を表示する

### その他の仕様は

- ID は 8 文字
- ID は 4 時間で切り替わる
- ID は掲示板ごとに異なる
